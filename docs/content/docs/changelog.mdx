## Version 2.4.0

Released: September 26, 2025

### üöÄ New Features

**Extended PlaceholderAPI Support**
- 60+ new placeholders for player statistics and server metrics
- Conditional placeholders with if_xxx_yes_no format
- Permission-based placeholders for dynamic content

**Enhanced Statistics System**
- Redesigned statistics tracking with comprehensive data collection
- Thread-safe statistics cache with performance metrics
- Cross-session persistent storage with automatic recovery
- Enhanced statistics book with 8 detailed categories

**Development Tools**
- Development environment detection for test commands
- Configuration validation and placeholder testing tools
- Automatic restriction of debug features to development builds

### üîß Improvements

**Configuration System**
- Automatic backup and recovery for corrupted configs
- Enhanced YAML validation with detailed error reporting
- Optimized configuration reload mechanisms

**Data Storage**
- Automatic error recovery for corrupted data files
- Timestamp-based backup system for data protection
- Improved cross-version compatibility handling

- Improved GUI cleanup when closing interfaces

- Added support for previewing Item shares directly when holding the item
