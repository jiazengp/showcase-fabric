# PlaceholderAPI Integration

<MinecraftVersion auto />

Showcase provides comprehensive PlaceholderAPI integration, allowing other mods and plugins to access player share data through standardized placeholders.

## Available Placeholders

### Player Share Information

<Tabs items={["Current Shares", "Share Statistics", "Permissions"]}>
  <Tab value="Current Shares">
    **Active Share Placeholders**:
    ```
    %showcase:shares_count%          # Number of active shares
    %showcase:shares_remaining%      # Shares until limit
    %showcase:last_share_type%       # Type of last share created
    %showcase:last_share_time%       # Time since last share
    %showcase:next_share_expires%    # When next share expires
    ```

    **Usage Examples**:
    ```
    Player has %showcase:shares_count% active shares
    Can create %showcase:shares_remaining% more shares
    Last shared: %showcase:last_share_type%
    ```
  </Tab>

  <Tab value="Share Statistics">
    **Lifetime Statistics**:
    ```
    %showcase:total_shares_created%  # Total shares ever created
    %showcase:total_shares_viewed%   # Total views received
    %showcase:most_shared_type%      # Most frequently shared type
    ```

    **Time-based Statistics**:
    ```
    %showcase:shares_today%          # Shares created today
    %showcase:shares_this_week%      # Shares created this week
    %showcase:average_share_duration% # Average duration of shares
    ```
  </Tab>

  <Tab value="Permissions">
    **Permission Check Placeholders**:
    ```
    %showcase:can_share_item%        # true/false
    %showcase:can_share_inventory%   # true/false
    %showcase:can_share_stats%       # true/false
    %showcase:can_use_chat_keywords% # true/false
    %showcase:has_admin_perms%       # true/false
    ```

    **Cooldown Information**:
    ```
    %showcase:item_cooldown%         # Seconds until can share item
    %showcase:inventory_cooldown%    # Seconds until can share inventory
    %showcase:chat_cooldown%         # Seconds until can use keywords
    ```
  </Tab>
</Tabs>

## Integration Examples

### Fabric Mod Integration

```java
public class ShowcasePlaceholderIntegration {

    public void getPlayerShareInfo(ServerPlayerEntity player) {
        // Use PlaceholderContext for Fabric placeholder API
        PlaceholderContext context = PlaceholderContext.of(player);

        // Parse placeholders in text
        Text shareCountText = Placeholders.parseText(
            Text.literal("You have %showcase:shares_count% active shares"),
            context
        );

        Text cooldownText = Placeholders.parseText(
            Text.literal("Item cooldown: %showcase:item_cooldown% seconds"),
            context
        );

        // Send messages to player
        player.sendMessage(shareCountText);
        player.sendMessage(cooldownText);
    }

    public void displayServerStats(ServerPlayerEntity player) {
        PlaceholderContext context = PlaceholderContext.of(player);

        Text serverStatsText = Placeholders.parseText(
            Text.literal("Server: %showcase:server_active_shares% active, %showcase:server_uptime% uptime"),
            context
        );

        player.sendMessage(serverStatsText);
    }
}
```

### Scoreboard Integration

```java
public class ShowcaseScoreboard {

    public void updateScoreboard(Player player, Scoreboard scoreboard) {
        Objective objective = scoreboard.getObjective("showcase");

        // Display share statistics
        String totalShares = PlaceholderAPI.setPlaceholders(player, "%showcase:total_shares_created%");
        String reputation = PlaceholderAPI.setPlaceholders(player, "%showcase:sharing_reputation%");

        Score shareScore = objective.getScore("§6Shares: §f" + totalShares);
        shareScore.setScore(15);

        Score repScore = objective.getScore("§6Reputation: §f" + reputation);
        repScore.setScore(14);
    }
}
```

## Advanced Placeholder Features

### Conditional Placeholders

<Tabs items={["Permission-based", "Time-based", "Share-based"]}>
  <Tab value="Permission-based">
    **Permission-conditional placeholders**:
    ```
    %showcase:if_can_share_item_yes_no%      # Returns "yes" or "no"
    %showcase:if_admin_admin_player%         # Returns "admin" or "player"
    %showcase:permission_level%              # Numeric permission level
    ```

    **Usage in messages**:
    ```
    You are currently a %showcase:if_admin_admin_player% user
    Item sharing: %showcase:if_can_share_item_enabled_disabled%
    ```
  </Tab>

  <Tab value="Time-based">
    **Time-conditional placeholders**:
    ```
    %showcase:if_cooldown_ready_waiting%     # "ready" or "waiting"
    %showcase:time_until_next_share%         # Formatted time string
    %showcase:shares_per_hour_limit%         # Rate limiting info
    ```

    **Formatted time outputs**:
    ```
    Next share available: %showcase:time_until_next_share%
    Status: %showcase:if_cooldown_ready_waiting%
    ```
  </Tab>

  <Tab value="Share-based">
    **Share-conditional placeholders**:
    ```
    %showcase:if_has_shares_active_none%     # "active" or "none"
    %showcase:most_viewed_share_type%        # Type with most views
    %showcase:longest_active_share%          # Duration of oldest share
    ```

    **Context-aware information**:
    ```
    Share status: %showcase:if_has_shares_active_none%
    Popular content: %showcase:most_viewed_share_type%
    ```
  </Tab>
</Tabs>

### Custom Placeholder Arguments

Some placeholders accept arguments for customization:

```
%showcase:cooldown:item%                 # Specific type cooldown
%showcase:shares_by_type:inventory%      # Count of specific type
%showcase:stats:today%                   # Statistics for specific period
%showcase:permission:showcase.admin%     # Check specific permission
```

## Configuration

### Enable PlaceholderAPI Support

```yaml title="config/showcase/config.yml"
integrations:
  placeholderapi:
    enabled: true

    # Update frequency for dynamic placeholders
    updateInterval: 30  # seconds

    # Cache placeholder results
    cacheResults: true
    cacheExpiration: 60  # seconds

    # Custom placeholder prefix (default: showcase)
    prefix: "showcase"
```

### Performance Settings

```yaml
placeholderapi:
  performance:
    # Async placeholder evaluation
    asyncEvaluation: true

    # Maximum concurrent evaluations
    maxConcurrentEvaluations: 10

    # Timeout for complex placeholders
    evaluationTimeout: 5000  # milliseconds
```

## Mod Integration Examples

### Economy Integration

```java
public class EconomyShowcaseIntegration {

    public void setupEconomyPlaceholders() {
        // Register custom placeholders
        PlaceholderAPI.registerPlaceholderHook("showcase_economy", new PlaceholderHook() {
            @Override
            public String onRequest(Player player, String params) {
                if (params.equals("sharing_earnings")) {
                    // Calculate earnings from sharing
                    int shares = Integer.parseInt(PlaceholderAPI.setPlaceholders(player, "%showcase:total_shares_created%"));
                    return String.valueOf(shares * 10); // 10 coins per share
                }
                return null;
            }
        });
    }
}
```

### Rank Integration

```java
public class RankIntegration {

    public String getPlayerRank(Player player) {
        String reputation = PlaceholderAPI.setPlaceholders(player, "%showcase:sharing_reputation%");
        int repScore = Integer.parseInt(reputation);

        if (repScore >= 1000) return "§6Master Sharer";
        if (repScore >= 500) return "§eExpert Sharer";
        if (repScore >= 100) return "§aActive Sharer";
        return "§7New Sharer";
    }
}
```

### Quest Integration

```java
public class QuestIntegration {

    public boolean checkQuestProgress(Player player, String questType) {
        switch (questType) {
            case "share_10_items":
                String itemShares = PlaceholderAPI.setPlaceholders(player, "%showcase:shares_by_type:item%");
                return Integer.parseInt(itemShares) >= 10;

            case "reach_reputation_100":
                String reputation = PlaceholderAPI.setPlaceholders(player, "%showcase:sharing_reputation%");
                return Integer.parseInt(reputation) >= 100;

            default:
                return false;
        }
    }
}
```

## Troubleshooting

### Common Issues

<Callout type="error">
**Placeholders not updating**: Check if PlaceholderAPI is installed and Showcase integration is enabled
</Callout>

**Solutions**:
1. Verify PlaceholderAPI is installed: `/papi version`
2. Check Showcase integration: `/showcase-manage status`
3. Reload PlaceholderAPI: `/papi reload`

<Callout type="error">
**Performance issues**: Too many placeholder evaluations causing lag
</Callout>

**Solutions**:
1. Enable caching in Showcase config
2. Reduce update intervals
3. Use async evaluation
4. Limit concurrent evaluations

```yaml
debug:

```
