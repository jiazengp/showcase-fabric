# Installation Guide

<MinecraftVersion auto />

Showcase is server-side only - players don't need to install anything!

## Requirements

<VersionRequirements showAllVersions />

**Required**:
- Fabric Loader - [Download from FabricMC](https://fabricmc.net/use/installer/)
- Fabric API - [Download from Modrinth](https://modrinth.com/mod/fabric-api)

**Optional**:
- LuckPerms - For advanced permissions ([Download](https://luckperms.net/download))

## Download Showcase

Get the latest version for your Minecraft version:

- **[Modrinth](https://modrinth.com/mod/showcase)** (Recommended)
- **[CurseForge](https://www.curseforge.com/minecraft/mc-mods/showcasemod)**
- **[GitHub Releases](https://github.com/jiazengp/showcase-fabric/releases)**

<Callout type="note">
Download the correct version for your Minecraft version - each MC version has its own build.
</Callout>

## Installation

1. **Stop your server**
2. **Place JAR files** in your `mods/` folder:
   - `fabric-api-[version].jar`
   - `showcase-[version].jar`
   - `luckperms-fabric-[version].jar` (optional)
3. **Start your server**

## Verify Installation

Check console for:
```log
[INFO] [showcase]: Showcase mod loaded successfully
```

Test with: `/showcase item` (while holding an item)

## Troubleshooting

**Common issues**:
- Wrong Minecraft version
- Missing Fabric API
- Fabric version mismatch

Check `logs/latest.log` for errors.
